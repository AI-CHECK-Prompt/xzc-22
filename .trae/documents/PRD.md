# 高校化学品全生命周期监管平台 — 产品需求文档（PRD）

## 1. 业务背景

高校化学与生物类实验室每日消耗的易制毒、易制爆、剧毒化学品与高活性试剂种类繁多。传统的电子表格 + 纸质单据模式在以下四个长期痛点上无解：

1. 采购计划与实际库存割裂，重复申购、跨学院调剂困难。
2. 双人双锁制度执行松散，事后补签难以追溯。
3. 废液归集跨期统计困难，倒桶交接无电子记录。
4. 审计追溯链路断裂，跨多张表手工拼接耗时易遗漏。

本平台从空仓库起步，覆盖化学品全生命周期，支撑院系日常运转与上级主管部门例行审计。

## 2. 业务目标

- 全流程数字化：采购、暂存、领用、配置、使用、回收、空瓶清洗、废液归集全部留痕。
- 监管合规：满足公安机关、生态环境部门、教育主管部门、保卫部门四方对接。
- 移动化：现场扫码与核验在移动端完成，支持离线缓存与自动同步。
- 高并发：20+ 院系、1000+ 实验室、10000+ 化学品的并发访问。

## 3. 角色与权限

| 角色 | 关键职责 |
| --- | --- |
| 设备秘书 | 提交采购申请、维护院系台账 |
| 导师 | 审批学生领用申请 |
| 项目负责人 | 审批项目组内领用 |
| 危化品管理员 | 终审与双人核验现场组织 |
| 实验人员 | 扫码登记用量、提交废液归集 |
| 保卫处 | 监督双人核验、查看追溯 |
| 科研处 | 审核采购计划 |
| 院系负责人 | 查看本院统计 |
| 主管部门（外审） | 只读审计追溯 |

## 4. 业务能力矩阵

| 模块 | 输入 | 处理 | 输出 |
| --- | --- | --- | --- |
| 采购计划 | 院系月度申购单 | 库存匹配 + 上限校验 + 三级审批 | 采购任务单 |
| 暂存台账 | 入库单 + 追溯码 | 唯一码生成（QR/RFID） | 可扫码试剂 |
| 领用审批 | 领用申请 | 三级审批 + 高危品类复核 | 电子领用单 |
| 双人核验 | 两名领用人签到 | 身份认证 + 定位 + 录像 | 核验记录 + 锁具联动 |
| 使用登记 | 扫码 + 表单 | 剩余量计算 + 阈值告警 | 使用流水 |
| 废液归集 | 扫码 + 类别选择 | 相容性提示 + 电子交接 | 废液桶流水 |
| 空瓶回收 | 扫码 | 清洗/销毁/厂家回收 | 回收记录 |
| 统计报表 | 全部业务流水 | 聚合分析 | 多维报表 |
| 审计追溯 | 追溯码 | 链路聚合 | 全链路视图 |

## 5. 合规对接

| 对接方 | 内容 |
| --- | --- |
| 公安机关易制毒化学品管理平台 | 年度许可、购买备案回执 |
| 生态环境部门危废管理计划备案系统 | 年度危废产生与处置数据 |
| 教育主管部门实验室安全年度报表 | 自动汇总 |
| 保卫部门门禁系统 | 实验室进出记录同步 |

> 对接以模拟适配器实现：保留真实接口契约，本地环境以 mock 数据演示完整流程。

## 6. 非功能要求

- 并发：≥ 500 在线用户，≥ 2000 QPS。
- 离线：移动端扫码 + 身份核验断网缓存 24 小时。
- 启动：从零到自检完成 ≤ 5 分钟。
- 审计：所有业务事件写入不可变流水，追溯链路在数据补录后保持完整。
